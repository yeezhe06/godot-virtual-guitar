# Godot Virtual Guitar 🎸

A simple guitar simulator made with Godot.

## Features

* 6 playable strings
* Click to pluck
* Drag to strum
* Keyboard controls
* Volume slider
* Animated string vibration

## Demo Video 🎥

[Watch Demo](https://www.youtube.com/watch?v=YOUR_VIDEO_ID)

## Controls

* Click strings
* Drag mouse across strings
* Press keys 1 to 6
* Press Q to Increase fret 
* Press A to Decrease fret


## How to Run

1. Open in Godot
2. Run project

## Built With

* Godot Engine
* GDScript
